# 天猫双十一美妆销售数据分析 | Tmall Double 11 Beauty Sales Analysis

## 项目简介
基于2016年天猫双十一美妆类目真实销售数据，对平台商品销量、销售额及品类结构进行探索性数据分析（EDA）。

在原有品类分析基础上，延伸探索了**价格区间与销量及销售额的关系**，
发现销量与销售额在不同价格段呈现明显分离，并提出对运营选品的业务建议。

## 数据说明
天猫双十一2016年美妆类目销售数据，包含商品名称、价格、销量、销售额、店铺、品类等字段。

## 主要分析内容
- 化妆品与护肤品大类的销量与销售额对比
- 二级品类（面霜、口红、精华等）销售表现
- 品牌店铺价格分布
- 延伸分析：价格区间与销量及销售额的关系

## 核心发现
销量与销售额呈明显分离：0-50元产品销量最高，但200-500元产品销售额贡献最大（约146亿）。
低价产品靠走量引流，中高端产品才是平台营收主力。
运营在资源分配上应向中高端品类倾斜。
